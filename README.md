### Installing

**Cloning the Repository**

```
$ git clone https://github.com/GabrielPoppes/AnimatedRainLoading

$ cd AnimatedRainLoading
```

**Installing dependencies**

```
$ yarn
```

_or_

```
$ npm install
```
